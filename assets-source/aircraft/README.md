# DHC-6 Float Model Source

`dhc-6_float.blend` is an independent working copy of the Blender file supplied
by the project owner on 2026-07-17. The original file remains unchanged in its
original OneDrive location.

The desktop app does not package the `.blend` file. It packages the optimized
GLB generated by `tools/blender_prepare_dhc6_float.py`.

The model contains an exterior airframe and floats but no cockpit interior,
transparent windshield, dashboard, or flight instruments. The desktop
simulator therefore uses this model for exterior views and provides live flight
instruments through its cockpit overlay.

Before redistributing the source or generated GLB outside the project, confirm
that the project has the required model redistribution rights.
