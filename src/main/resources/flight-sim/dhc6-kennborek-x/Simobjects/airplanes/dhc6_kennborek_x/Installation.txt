INSTALLATION for FSX:

Extract *ALL* the files in the zip to the FSX main folder. Not the simobjects folder, not the airplanes folder.

This will automatically create new folders and populate them with the correct files.

Users of Windows Vista and WIN7 may have to temporarily disable the Windows User Account Control (UAC) before installation to allow the creation of new folders.

The first time you load the model in FSX several pop-up panels will apppear asking:
"Are you sure you wish to run this software?"
Click the RUN and YES buttons in every case.
